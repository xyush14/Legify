Why Headnote — the moat

Three-check verification.
Built so you never quote a fake case.

Generic AI tools hallucinate citations. We don't — by design. Every output passes three independent checks before it reaches your screen.

CHECK 01

Case ID exists

Every cited case is matched against the curated corpus or Indian Kanoon. If the case doesn't exist, the output is regenerated automatically.

CHECK 02

Paragraph anchor is real

The cited paragraph number is checked against the actual judgment text. No "para 47" that turns out to be para 23 of a different case.

CHECK 03

Quoted phrase is verbatim

Any text in quotes must appear verbatim in the source paragraph. Paraphrases are flagged. No misattributed Bench observations.
